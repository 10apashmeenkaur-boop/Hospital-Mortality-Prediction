#This project investigates the primary drivers of in-hospital mortality for admitted patients by integrating SQL data engineering with Tableau visualization. The analysis identifies critical correlations between patient demographics, clinical vitals, and pre-existing comorbidities to improve clinical decision-making.

Project Workflow
1)Data Preprocessing: Raw datasets underwent initial cleaning and structuring in Excel to ensure data integrity before database ingestion.


2)Database Management: The refined data was imported into a MySQL environment for structured storage and high-performance querying.


3)Exploratory Data Analysis (SQL): Advanced SQL queries were executed to segment and analyze patient attributes, including:
-Demographics: Age, ethnicity, and gender distributions.
-Biometrics: Weight, BMI, and heart rate fluctuations.
-Clinical Indicators: Presence and impact of specific comorbidities on survival rates.


4)Data Visualization: Findings were translated into a comprehensive Tableau dashboard, providing an interactive interface to monitor trends and mortality risk factors.
